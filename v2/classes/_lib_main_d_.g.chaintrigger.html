fire

  • fire(arg?: T): void

  • Inherited from Trigger.fire

    • Defined in lib/main.d.ts:1261

    このTriggerを発火する。

    登録された全ハンドラの関数を、引数 arg を与えて呼び出す。 呼び出し後、次のいずれかの条件を満たす全ハンドラの登録は解除される。

    • ハンドラが addOnce() で登録されていた場合
    • ハンドラが add() で登録される際に once: true オプションが与えられていた場合
    • 関数がtruthyな値を返した場合

    Parameters

    • Optional arg: T

      ハンドラに与えられる引数

    Returns void
