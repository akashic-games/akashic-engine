Interface ScriptAssetRuntimeValue

ScriptAsset の実行時、g 以下に加えられる値を定めたinterface。 g の実際の値は、本来の g のすべてのプロパティに加えて以下を持つ必要がある。

通常のゲーム開発者がこのクラスを直接利用する必要はない。 ScriptAsset を実行する場合は、暗黙にこのクラスを利用する require() を用いるべきである。

interface ScriptAssetRuntimeValue {
    dirname: string;
    exports: any;
    filename: string;
    game: any;
    module: Module;
}
