Namespace EventIndex

@akashic/playlog の各 interface に名前を与える const enum 群。

playlog.Tick や playlog.Event は、実態としてはただのヘテロ配列である。 各 interface にはインデックスと型名のみがあり、それぞれの値の意味内容は自然言語でしか記述されていない。 インデックスのハードコーディングを避けるため、ここで const enum で名前を与えることにする。

本当はこのファイルの内容は playlog に移管すべきだが、 playlog に存在しない Local フィールドを使うため akashic-engine 側で扱う。
